# Guía absoluta para principiantes: Programación en Python — Edición en español

Esta carpeta contiene los archivos de apoyo para la edición en español de *Absolute Beginner’s Guide to Python Programming*.

## Estructura

Los archivos están organizados por capítulo:

- `Chapter 02 - Los fundamentos de Python`
- `Chapter 03 - Explorar Python con bucles y colecciones`
- `Chapter 04 - Funciones módulos paquetes y lambdas`
- `Chapter 05 - Almacenamiento de datos archivos de texto y JSON`
- `Chapter 06 - Programación orientada a objetos`
- `Chapter 07 - Interfaces gráficas con Tkinter`

Dentro de cada capítulo puedes encontrar una o más de estas carpetas:

- `Examples`: ejemplos cortos relacionados con el capítulo.
- `Exercises`: ejercicios prácticos.
- `Project - ...`: proyecto principal del capítulo.

## Cómo usar estos archivos

Puedes descargar este repositorio como archivo ZIP desde GitHub. No necesitas una cuenta de GitHub para hacerlo.

Después de descargarlo, abre la carpeta del capítulo correspondiente y ejecuta los archivos `.py` con Python.

Ejemplo:

```bash
python main.py
```

Algunos proyectos crean archivos temporales, como `gastos.json`, mientras se ejecutan. Es normal; esos archivos almacenan los datos que ingresas durante la práctica.

## Nota

Los nombres de carpetas están en español para que coincidan con la edición traducida. Algunos nombres de archivos o módulos pueden mantenerse en inglés cuando ayudan a conservar compatibilidad con imports o prácticas comunes de Python.
