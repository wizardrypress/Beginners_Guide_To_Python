# Proyecto: Calculadora de pagos de préstamos

Este proyecto corresponde a la sección **2.16 Proyecto: Calculadora de pagos de préstamos** de la edición en español 2026.

## Cómo ejecutarlo

Abre una terminal en esta carpeta y ejecuta:

```bash
python main.py
```

Algunos proyectos pueden crear archivos de datos mientras se ejecutan. Eso es normal.
