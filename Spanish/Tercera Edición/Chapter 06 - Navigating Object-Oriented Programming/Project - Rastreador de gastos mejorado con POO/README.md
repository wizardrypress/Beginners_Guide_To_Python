# Proyecto: Rastreador de gastos mejorado con POO

Este proyecto corresponde a la sección **6.16 Proyecto del capítulo: Rastreador de gastos mejorado con POO** de la edición en español 2026.

## Cómo ejecutarlo

Abre una terminal en esta carpeta y ejecuta:

```bash
python main.py
```

Algunos proyectos pueden crear archivos de datos mientras se ejecutan. Eso es normal.
