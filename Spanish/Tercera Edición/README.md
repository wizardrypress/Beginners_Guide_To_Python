# Recursos de la Tercera Edición / 2026 en español

Esta carpeta contiene los archivos de apoyo para la **Tercera Edición / 2026** en español de **Guía absoluta para principiantes: Programación en Python**.

## Cómo encontrar los archivos

Cada carpeta de capítulo puede incluir:

- `Exercises` — soluciones de los ejercicios del capítulo.
- `Project - ...` — solución completa del proyecto del capítulo.
- `README.md` — guía rápida de lo que contiene la carpeta.

Para ejecutar la mayoría de los proyectos, abre una terminal dentro de la carpeta del proyecto y ejecuta:

```bash
python main.py
```
