# Capítulo 3: Explorar Python con bucles y colecciones

Esta carpeta contiene los archivos de apoyo para la edición en español 2026.

## Qué contiene

- `Exercises` — soluciones de los ejercicios de la sección **3.14 Ejercicios del capítulo**.
- `Project - Juego de preguntas` — solución del proyecto.

## Cómo ejecutar los archivos

Para ejecutar un ejercicio, abre una terminal dentro de la carpeta `Exercises` y usa:

```bash
python nombre_del_archivo.py
```

Para ejecutar un proyecto, abre una terminal dentro de la carpeta del proyecto y usa:

```bash
python main.py
```
