# Proyecto: Juego de preguntas

Este proyecto corresponde a la sección **3.15 Proyecto: Crear un juego de preguntas en Python** de la edición en español 2026.

## Cómo ejecutarlo

Abre una terminal en esta carpeta y ejecuta:

```bash
python main.py
```

Algunos proyectos pueden crear archivos de datos mientras se ejecutan. Eso es normal.
