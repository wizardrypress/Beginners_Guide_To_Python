# Proyecto: Juego del gato con IA

Este proyecto corresponde a la sección **4.14 Presentación de un adversario IA / 4.15 Guía paso a paso para construir la IA** de la edición en español 2026.

## Cómo ejecutarlo

Abre una terminal en esta carpeta y ejecuta:

```bash
python main.py
```

Algunos proyectos pueden crear archivos de datos mientras se ejecutan. Eso es normal.
