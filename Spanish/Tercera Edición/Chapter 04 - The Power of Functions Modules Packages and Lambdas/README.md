# Capítulo 4: El poder de las funciones, módulos, paquetes y lambdas

Esta carpeta contiene los archivos de apoyo para la edición en español 2026.

## Qué contiene

- `Exercises` — soluciones de los ejercicios de la sección **4.10 Ejercicios del capítulo**.
- `Project - Juego del gato` — solución del proyecto.
- `Project - Juego del gato con IA` — solución del proyecto.
- `Project - Competencia MiniMax` — solución del proyecto.

## Cómo ejecutar los archivos

Para ejecutar un ejercicio, abre una terminal dentro de la carpeta `Exercises` y usa:

```bash
python nombre_del_archivo.py
```

Para ejecutar un proyecto, abre una terminal dentro de la carpeta del proyecto y usa:

```bash
python main.py
```
