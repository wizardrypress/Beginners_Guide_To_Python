# Proyecto: Generador de recibos localizados

Este proyecto corresponde a la sección **7.14 Proyecto del capítulo: Generador de recibos localizados** de la edición en español 2026.

## Cómo ejecutarlo

Abre una terminal en esta carpeta y ejecuta:

```bash
python main.py
```

El programa te pedirá un código de idioma:

- `en` — inglés
- `es` — español
- `pt` — portugués

Si escribes un código no compatible, el programa usará inglés como alternativa.
