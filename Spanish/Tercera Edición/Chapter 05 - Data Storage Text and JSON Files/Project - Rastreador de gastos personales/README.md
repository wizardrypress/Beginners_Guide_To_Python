# Proyecto: Rastreador de gastos personales

Este proyecto corresponde a la sección **5.10 Proyecto del capítulo: Rastreador de gastos personales** de la edición en español 2026.

## Cómo ejecutarlo

Abre una terminal en esta carpeta y ejecuta:

```bash
python main.py
```

Algunos proyectos pueden crear archivos de datos mientras se ejecutan. Eso es normal.
