# Selector de edición en español

Elige la carpeta que coincida con la edición impresa en la portada de tu libro.

| Edición del libro | Carpeta |
|---|---|
| Tercera Edición / 2026 | [`Tercera Edición/`](Tercera%20Edici%C3%B3n/) |
| 2025 | [`2025/`](2025/) |

Si tu libro dice **Tercera Edición** o **2026**, usa [`Tercera Edición/`](Tercera%20Edici%C3%B3n/).

La carpeta [`2026/`](2026/) solo existe como redirección para lectores que buscan la edición por año.
