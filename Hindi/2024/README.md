# Bilkul Beginners ke liye Python Programming Guide — Hindi/Hinglish Edition 2024

Yeh folder Hindi/Hinglish edition ke liye source code, examples, exercises, aur chapter projects rakhta hai.

## Folder structure

Files chapter ke hisaab se organize kiye gaye hain:

- `Chapter 01 - Prarambhik charan`
- `Chapter 02 - Python ki neev`
- `Chapter 03 - Loops aur collections`
- `Chapter 04 - Functions modules packages aur lambdas`
- `Chapter 05 - Text aur JSON files`
- `Chapter 06 - Object oriented programming`
- `Chapter 07 - Tkinter GUI`

Har chapter mein inme se kuch folders ho sakte hain:

- `Examples` — chapter se related chhote examples.
- `Exercises` — practice exercise solutions.
- `Project - ...` — chapter ka complete project.

## Code kaise run karein

Pehle code download karein:

1. GitHub page par green **Code** button par click karein.
2. **Download ZIP** select karein.
3. ZIP file ko apne computer par unzip karein.
4. Is folder ko open karein:

```text
Hindi/2024
```

Python file run karne ke liye terminal ya command prompt mein us file ke folder tak jaayein aur run karein:

```bash
python file_name.py
```

Agar project mein multiple files hain, to project folder mein jaakar `main.py` run karein:

```bash
python main.py
```

## Note

Folder aur file names Roman-script Hinglish mein rakhe gaye hain taaki Windows, GitHub, aur Python imports ke saath encoding problems kam ho. Code ke andar prompts, comments, aur printed messages beginner-friendly Hinglish mein rakhe gaye hain.

Kuch Python identifiers ya module names English mein reh sakte hain. Yeh jaan-boojhkar kiya gaya hai, kyunki Python ecosystem, documentation, aur online examples mein English names common hote hain.
