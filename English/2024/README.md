# English 2024 Edition Resources

This folder contains the support files for the English 2024 edition.

The 2024 and 2025 English editions use the same core chapter support files. This folder is provided for readers with the 2024 paperback so they can open the edition printed on their book.

## How to Find Files

Each chapter folder may contain:

- `Exercises` — solutions for the chapter's practice exercises.
- `Project - ...` — the completed chapter project.
- `Examples` — short example programs, when available.

Most projects should be run from inside the project folder using:

```bash
python main.py
```
