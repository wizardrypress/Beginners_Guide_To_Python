## Why There Are No Code Files Here

The exercises in this chapter are different from the rest of the book. Instead
of writing a program from scratch, each exercise asks you to practice using AI
as a thinking partner — the augmentation approach described throughout the
chapter.

There are no code solutions because there are no single correct answers. The
value comes from the process itself:

- **Exercise 1: Revisit and Explain** — Go back to a previous chapter, re-solve
  an exercise, and explain your code to an AI line by line to discover what you
  truly understand.
- **Exercise 2: Debug with Intention** — Deliberately introduce a bug into a
  working program and practice reasoning through the error with AI guidance.
- **Exercise 3: Extend Beyond the Book** — Pick a chapter project and add a new
  feature using AI as a thinking partner, not a code generator.

These exercises reflect how professional developers actually work — thinking
critically, debugging methodically, and building on what already exists.
