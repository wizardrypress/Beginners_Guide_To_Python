# Chapter 5 Resources

This folder contains the GitHub resources for **Chapter 5** of the English 2026 edition.

- **5.9 Chapter Exercises** → `Exercises/`
- **5.10 Chapter Project: Personal Expense Tracker** → project folder listed below

## Folder contents

- `Exercises` — Solutions for the Chapter 5 practice exercises.
- `Project - Personal Expense Tracker` — Completed solution for the personal expense tracker project.

## Running a project

Open the project folder first, then run:

```bash
python main.py
```

Run exercise files from inside the `Exercises` folder, for example:

```bash
python exercise_1_basic_variable_assignment.py
```
