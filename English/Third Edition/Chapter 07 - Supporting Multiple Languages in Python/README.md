# Chapter 7 Resources

This folder contains the GitHub resources for **Chapter 7** of the English 2026 edition.

- **7.13 Chapter Exercises** → `Exercises/`
- **7.14 Chapter Project: Localized Receipt Generator** → project folder listed below

## Folder contents

- `Exercises` — Solutions for the Chapter 7 localization exercises.
- `Project - Localized Receipt Generator` — Completed solution for the localized receipt generator project.

## Running a project

Open the project folder first, then run:

```bash
python main.py
```

Run exercise files from inside the `Exercises` folder, for example:

```bash
python exercise_1_basic_variable_assignment.py
```
