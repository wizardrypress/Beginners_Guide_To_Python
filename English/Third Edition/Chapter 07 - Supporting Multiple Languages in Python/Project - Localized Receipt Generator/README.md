# Project 7.14: Localized Receipt Generator

This folder contains the completed solution for **Chapter 7, Project 7.14: Localized Receipt Generator**.

## What the project demonstrates

- Storing translated user-facing text in dictionaries
- Selecting a language using a language code
- Falling back to English when an unsupported language code is entered
- Printing a receipt with translated labels
- Formatting today's date with `strftime()`

## How to run it

From this folder, run:

```bash
python main.py
```

Then enter one of these language codes:

- `en` for English
- `es` for Spanish
- `pt` for Portuguese

Try entering an unsupported code, such as `fr`, to see the English fallback behavior.
