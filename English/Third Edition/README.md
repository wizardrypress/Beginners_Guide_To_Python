# English Third Edition / 2026 Resources

This folder contains the support files for the English **Third Edition / 2026** edition.

## How to Find Files

Each chapter folder may contain:

- `Exercises` — solutions for the chapter's practice exercises.
- `Project - ...` — the completed chapter project.
- `Examples` — short example programs, when available.

Most projects should be run from inside the project folder using:

```bash
python main.py
```


## Quick Path Examples

Looking for a specific exercise or project? Start in this folder, then open the chapter folder.

- Project 3.15: `Chapter 03 - Exploring Python with Loops and Collections/Project - Quiz Game/main.py`
- Project 7.14: `Chapter 07 - Supporting Multiple Languages in Python/Project - Localized Receipt Generator/main.py`
