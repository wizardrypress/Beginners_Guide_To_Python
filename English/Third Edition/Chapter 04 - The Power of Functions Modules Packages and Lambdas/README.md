# Chapter 4 Resources

This folder contains the GitHub resources for **Chapter 4** of the English 2026 edition.

- **4.10 Chapter Exercises** → `Exercises/`
- **4.11 Project: Tic-Tac-Toe Game** → project folder listed below

## Folder contents

- `Exercises` — Solutions for the Chapter 4 practice exercises.
- `Project - Tic-Tac-Toe` — Human vs. human version of the tic-tac-toe project.
- `Project - Tic-Tac-Toe with AI` — Version of the tic-tac-toe project with an AI opponent.
- `Project - MiniMax Competition` — Advanced AI/minimax exploration project.

## Running a project

Open the project folder first, then run:

```bash
python main.py
```

Run exercise files from inside the `Exercises` folder, for example:

```bash
python exercise_1_basic_variable_assignment.py
```
