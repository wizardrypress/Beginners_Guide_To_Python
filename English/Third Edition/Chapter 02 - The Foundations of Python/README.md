# Chapter 2 Resources

This folder contains the GitHub resources for **Chapter 2** of the English 2026 edition.

- **2.15 Chapter Exercises** → `Exercises/`
- **2.16 Project: Loan Payment Calculator** → project folder listed below

## Folder contents

- `Exercises` — Solutions for the Chapter 2 practice exercises.
- `Project - Loan Calculator` — Completed solution for the loan payment calculator project.

## Running a project

Open the project folder first, then run:

```bash
python main.py
```

Run exercise files from inside the `Exercises` folder, for example:

```bash
python exercise_1_basic_variable_assignment.py
```
