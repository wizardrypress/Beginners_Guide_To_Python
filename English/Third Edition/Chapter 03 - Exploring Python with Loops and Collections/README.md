# Chapter 3 Resources

This folder contains the GitHub resources for **Chapter 3** of the English 2026 edition.

- **3.14 Chapter Exercises** → `Exercises/`
- **3.15 Project: Build a Python Quiz Game** → project folder listed below

## Folder contents

- `Exercises` — Solutions for the Chapter 3 practice exercises.
- `Project - Quiz Game` — Completed solution for the Python quiz game project.

## Running a project

Open the project folder first, then run:

```bash
python main.py
```

Run exercise files from inside the `Exercises` folder, for example:

```bash
python exercise_1_basic_variable_assignment.py
```
