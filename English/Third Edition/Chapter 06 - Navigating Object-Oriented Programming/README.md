# Chapter 6 Resources

This folder contains the GitHub resources for **Chapter 6** of the English 2026 edition.

- **6.15 Chapter Exercises** → `Exercises/`
- **6.16 Chapter Project: Enhanced Expense Tracker with OOP** → project folder listed below

## Folder contents

- `Exercises` — Solutions for the Chapter 6 practice exercises.
- `Project - Enhanced Expense Tracker with OOP` — Completed OOP version of the expense tracker project.

## Running a project

Open the project folder first, then run:

```bash
python main.py
```

Run exercise files from inside the `Exercises` folder, for example:

```bash
python exercise_1_basic_variable_assignment.py
```
