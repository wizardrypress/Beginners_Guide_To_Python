## About This Repository

This repository contains all of the source code, exercise solutions, and chapter
project files referenced in the book. It is organized by language and edition so
you can find the code that matches your copy of the book.

You do not need a GitHub account to download these files. Click the green **Code**
button above, then select **Download ZIP**.

## How to Use This Repository

Navigate to the folder that matches your edition. Inside, each chapter folder
contains:

- **Exercises/** — Solutions to the chapter exercises found in the book.
  These match the exercise solutions in Appendix A.
- **Project/** — Complete source code for the chapter's capstone project.
  the code here is the finished result.

Exercise solutions are provided so you can check your work. We recommend
attempting each exercise on your own before looking at the solution.

## What's New in the 2026 English Edition

The 2026 edition has been updated with two new chapters, expanded content, and
a reorganized structure.

**New chapters:**
- **Chapter 7: Supporting Multiple Languages in Python (Localization)** —
  Learn how to build software that supports users around the world with
  translated messages, locale-aware number formatting, and regional date
  displays.
- **Chapter 8: Using AI to Build In-Demand Skills and Become Unstoppable** —
  Learn how to use AI tools as a thinking partner to accelerate your
  learning without sacrificing real understanding.

**New content in existing chapters:**
- Chapter 1 expanded with a section on programming and the English language
- Chapter 2 expanded with sections on variable names and language, and
  variables vs parameters vs arguments
- Chapter 6 expanded with a section on what OOP really unlocks

**Removed from the print edition (available as bonus PDFs):**
- Graphical User Interfaces with Tkinter (was Chapter 7 in the 2025 edition)
- GitHub Primer for Downloading Source Code (was Appendix B)
- Choosing the Right IDE (was Appendix D)

## Bonus Chapters

Some content from previous editions was removed from the 2026 print edition to
make room for new material. This content is still valuable, so we have made it
available as free PDF downloads in the `Bonus Chapters/` folder within each
edition.

| Bonus Chapter | Description |
|---------------|-------------|
| Graphical User Interfaces with Tkinter | Build desktop applications with Python's built-in GUI toolkit. Covers layout managers, widgets, event handling, and includes a project to convert the Tic-Tac-Toe game into a graphical application. |
| GitHub Primer for Downloading Source Code | A step-by-step guide to downloading code from GitHub, for readers who are new to the platform. |
| Choosing the Right IDE | A comparison of popular Python development environments including VS Code, PyCharm, and Jupyter Notebook. |

## Python Version

The code in this repository is written for **Python 3.10 or later**. Most examples
will work with Python 3.8+, but we recommend using the latest stable version of
Python 3.
