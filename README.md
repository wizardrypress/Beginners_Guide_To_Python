## How to Download the Code

You do **not** need a GitHub account.

1. Click the green **Code** button near the top of this GitHub page.
2. Click **Download ZIP**.
3. Unzip the downloaded file on your computer.
4. Open the folder that matches your language and edition.
5. Open the chapter folder you are working on.

## Python Version

The code in this repository is written for **Python 3.10 or later**.

Most examples will also work with Python 3.8+, but the latest stable version of Python 3 is recommended.
